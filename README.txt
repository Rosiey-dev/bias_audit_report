README: Running the Notebook
🔧 Requirements
For VS Code
• Visual Studio Code (https://code.visualstudio.com/)
• Python (preferably 3.7+)
• Jupyter extension for VS Code
• Required Python libraries (see dependencies section)
For Google Colab
• A Google account
• Web browser
📁 Files
• Untitled0.ipynb: The main Jupyter notebook
🚀 Running in VS Code
1. Install Requirements
   - Install Python from python.org
   - Install VS Code
   - Open VS Code and install the Jupyter and Python extensions

2. Open the Notebook
   - Launch VS Code
   - Open the folder containing Untitled0.ipynb
   - Open the file in interactive mode

3. Create or Select a Python Environment
   - Use Ctrl+Shift+P > “Python: Select Interpreter”

4. Install Required Libraries
   - Open a terminal in VS Code and run:
     pip install -r requirements.txt

5. Run the Notebook
   - Use the “Run All” button or run cells with Shift+Enter
   
☁️ Running in Google Colab

1. Open the Notebook in Colab
   - Go to https://colab.research.google.com/
   - Click “Upload” and select Untitled0.ipynb

2. Run the Cells
   - Click “Runtime” > “Run all”

3. Install Additional Libraries (if needed)
   - Use !pip install <package> in a code cell
📦 Dependencies
To ensure smooth execution, install the following Python libraries:

pip install numpy pandas matplotlib seaborn scikit-learn
